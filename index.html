Local: http://localhost:3001 · Producción: https://tuapi.com
Pasos para activar
1 Descarga los archivos del servidor desde el chat de desarrollo
2 Renombra .env.example a .env
3 Pega tu API Key de console.anthropic.com
4 Ejecuta npm install y luego npm start
5 Ingresa la URL aquí y haz click en Verificar conexión
📱 Instalar como App
Android (Chrome)
  1. Abre esta página en Chrome
  2. Toca los 3 puntos arriba a la derecha
  3. Selecciona "Agregar a pantalla de inicio"
  4. Confirma → ya tienes el ícono como app
iPhone (Safari)
  1. Abre en Safari (no Chrome)
  2. Toca el ícono de compartir (cuadrado con flecha)
  3. Selecciona "Añadir a inicio"
  4. Confirma → ya aparece en tu pantalla
💡 Una vez instalada funciona sin internet y se ve como app nativa sin barra del navegador.
